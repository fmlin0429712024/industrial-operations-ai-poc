# Model Artifact

`esp_risk_model.joblib` is the small, synthetic-only model artifact generated by this POC's training pipeline. It is intentionally versioned here because it contains no client data, no client IP, and is only 3.7 KB.

| Property | Value |
|---|---|
| Model | Logistic regression |
| Prediction | ESP-related intervention or material production-loss risk within 30 days |
| Input | 10 production, ESP telemetry, and operating-context features |
| Review threshold | 0.60 |
| Data | Fully synthetic |

The artifact stores the fitted model, ordered feature list, and selected threshold. `src/score_case.py` loads it and calls `predict_proba()` during inference.

This Git-tracked artifact is appropriate only for a public-safe POC. A real client model would be versioned in a governed model registry with approval, lineage, retention, and access controls.
