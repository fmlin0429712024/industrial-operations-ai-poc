# Upstream ESP Reliability Trial Scope

This folder holds the customer-facing trial-scoping deliverables built from the public-safe Industrial Operations AI POC.

## Scope

- **Industry:** upstream oil and gas production
- **Chosen trial:** early identification of electric submersible pump (ESP) failure or material production-loss risk
- **Decision:** provide a ranked, evidence-backed risk signal for production engineers; retain human approval before any field action

## Deliverables

- `deliverables/` — finalized PowerPoint and PDF submission artifacts
- `source/` — presentation source and concise assumptions used in the trial narrative

## Boundary

All figures, client assumptions, and operational examples in this folder are illustrative. The work is a scoping exercise, not a claim of a live deployment or a production control system.
